# Unifacef React js

Aplicação feita em React js.

# Criar uma aplicação
```bash
npx create-react-app nome-do-app --template typescript
```

# Instalar dependências
```
npm i -g yarn
yarn
```

# Compilar a aplicação
```bash
yarn build
```

# Configurar o servidor estático
```bash
  npm i serve --g
  serve -s build
```

# Iniciar a aplicação
```
yarn start
```

# Rodar os testes
```
yarn test
```
